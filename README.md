## Installation

    npm install

## How to run

For development:

    npm run dev
    open http://localhost:8082

For dev prod:

    npm run prod

## Deploying

Build:
    
    npm run build

Start:

    npm run start
    open http://localhost:8050
    
Deploy:

    now

## Requirements

    Node 8+
